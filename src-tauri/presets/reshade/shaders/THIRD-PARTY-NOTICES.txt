ReShade shader collection (reshade-shaders)
===========================================

Game Settings Master includes HLSL/FX shader files from the community
ReShade shader repository:

  https://github.com/crosire/reshade-shaders
  Branch used by GSM: nvidia

Each .fx file may carry its own copyright and license in the file header.
GSM universal presets (performance / clarity / cinematic) use at minimum:

  Clarity.fx
    Author: Ioxa
    (see file header in Shaders/Clarity.fx)

  Vignette.fx
    Author: Christian Cann Schuldt Jensen (CeeJay.dk)
    (see file header in Shaders/Vignette.fx)

  AdaptiveSharpen.fx
    Copyright (c) 2015-2018, bacondither
    BSD-style license — full text in Shaders/AdaptiveSharpen.fx

Other shaders in presets/reshade/shaders/Shaders/ are included for ReShade
compatibility; retain their original headers when redistributing.

Game Settings Master does not claim ownership of these shaders.
